Create a GitHub Personal Access Token

  1. Open GitHub Settings and go to Developer settings.
  2. Select Personal access tokens, then choose Tokens (classic) or Fine-grained tokens.
  3. Create a token with read-only repository access.
  4. Copy the token and paste it into the Personal Access Token field.
